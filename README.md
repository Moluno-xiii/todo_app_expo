# Todo Application

Built for HNG 13 frontend stage 3b task.
[Figma File](https://www.figma.com/design/NRbd5hcrQcAa1LBbctUhf9/todo-app?node-id=0-335&t=xM2WMmacWyAoTWtn-0)

## Features

- Dynamic and persistent Theme switching
- Basic CRUD on for todos
- DB storage on cart checkout (with convex)

## Technologies used

- [React native](https://reactnative.dev/docs/getting-started)
- [Convex](https://docs.convex.dev/)
- [Typescript](https://www.typescriptlang.org/)
- [Expo](https://docs.expo.dev/)

- Download 'Expo go' app from your app store

```bash

cd todo_app_expo
npm install
npm start
```
